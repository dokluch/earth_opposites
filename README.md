# 🌍 Earth Opposites — Antipodal Explorer

**What's on the exact opposite side of the world from you?**

An interactive app that lets you explore antipodal points — places that are diametrically opposite each other on Earth. Click anywhere on a map, drag points on a 3D globe, or browse 120+ curated city pairs.

## ✨ Features

- **Dual interactive maps** — OpenStreetMap-powered panels for both your point and its antipode
- **3D globe** — d3-geo orthographic projection with Natural Earth land boundaries, drag-to-rotate, and draggable points
- **120+ antipodal city pairs** — curated pairs like Wellington ↔ Salamanca, Shanghai ↔ Buenos Aires
- **Real-time geocoding** — reverse geocoding via Nominatim shows place names as you click
- **Search** — filter city pairs by name or country
- **Random pair on load** — every visit starts with a surprise

## 🛠 Stack

React 19 · TypeScript · Vite · MapLibre GL · d3-geo · world-atlas · Nominatim

## 🚀 Run it

```bash
npm install
npm run dev
```
